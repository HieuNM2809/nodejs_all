## Technology stack

- NodeJS
- ExpressJS
- Mongoose

## Requirements

- Node >= 14.19.1
- yarn

## Running App

- set node version using

```
    nvm
```

- install dependency

```
    yarn
```

- Copy and rename .env.example file to .env

- Run it with sandbox environment

```
    yarn dev
```

## More

Application running on port [http://localhost:8800](http://localhost:8800).
