## Technology stack

- ReactJS
- Redux
- PeerJS

## Requirements

- Node >= 14.19.1
- yarn

## Running App

- set node version using

```
    nvm
```

- install dependency

```
    yarn
```

- Run it with sandbox environment

```
    yarn dev
```

## More

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
