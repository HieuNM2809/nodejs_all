Hướng dẫn chạy dự án:
 
- Socket: 
	+ Cài đặt các dependency bằng lệnh "yarn"
	+ Khởi chạy sever socket bằng lệnh "yarn dev"
	-> Ứng dụng socket được chạy với địa chỉ http://localhost:8900
 
- Backend(Technology stack: NodeJS,ExpressJs,Mongoose):
	+ set node version: sử dụng lệnh "nvm"
	+ Cài đặt các dependency bằng lệnh "yarn"
	+ Copy file .env.example ra và đổi tên file vừa copy thành .env
	+ Khởi chạy app bằng lệnh "yarn dev"
	-> Backend khởi chạy với địa chỉ http://localhost:8800
 
- Frontend(Technology stack: ReactJS, Redux, PeerJS):
	+ set node version: sử dụng lệnh "nvm"
	+ Cài đặt các dependency bằng lệnh "yarn"
	+ Khởi chạy app bằng lệnh "yarn dev"
	-> Kết quả Browser được chạy trên địa chỉ http://localhost:3000
