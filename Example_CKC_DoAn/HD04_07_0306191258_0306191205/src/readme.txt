Đầu tiên là việc cài đặt node_modules:
	Để cài node_modules cho folder frontend dùng lệnh: npm install
	Để cài node_modules cho folder backend dùng lệnh: yarn install

Tiếp theo là chạy project:
	Để chạy project frontend dùng lệnh: npm start
	Để chạy project backend dùng lệnh: yarn start  yarn start-dev
