HƯỚNG DẪN SỬ DỤNG SOURCE QMSNET-FE

1. Download source
- Git Repository: https://github.com/Quan-Kiu/QMSNet-FE
2. Cài đăt package
- Mở command line đi tới folder chứa souce code.
- Chạy lệnh npm i.
3. Cấu hình biến môi trường
- Copy file .env.example thành một file có tên .env.
- Cấu hình các biến môi trường tương ứng.
4. Chạy project
- Chạy lệnh npm start
